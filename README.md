# detect-regex
Detect regex patterns in strings

index.js
```javascript
import Detector from "detect-regex";
const detector = new Detector();
detector.print();
```

tsconfig.json
```json
{
  "compilerOptions": {
    "target": "ES6",
    "module": "ES6",
    "esModuleInterop": true,
    "moduleResolution": "node",
  }
}
```
